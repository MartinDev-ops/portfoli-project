## Bootstrap Portfolio 

## Overview

This is a responsive personal portfolio website built using **HTML**, **CSS**, and **Bootstrap 5**. The project was completed as part of the Zaio Full Stack/AI March Bootcamp to demonstrate responsive web design and Bootstrap fundamentals.

## Features

* Responsive Hero Section
* Sticky Navigation Bar
* About Me Section
* Projects Grid
* Contact Form
* Footer
* Mobile-friendly design

## Technologies Used

* HTML
* CSS
* Bootstrap 5
* Bootstrap Icons
